
# Remove-Applibappvserver
Removes an App-V server information
## Syntax

```
Remove-AppLibAppVServer [-Uid] <Int32> [-LoggingId <Guid>] [-BearerToken <String>] [-TraceParent <String>] [-TraceState <String>] [-VirtualSiteId <String>] [-AdminAddress <String>] [<CommonParameters>]
```

## Detailed Description
Provides the ability to remove an App-V server, associated packages and applications.
